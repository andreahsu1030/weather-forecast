## 專案說明

這是一個用於顯示天氣預報的應用，使用了台灣中央氣象局的開放數據 API。使用者可以查看當前天氣、日出日落、月出月落，以及未來一週的天氣預測。

專案連結：https://andreahsu1030.github.io/weather-forecast/

## 功能介紹
+ 顯示當前天氣狀況，包括溫度、濕度、降雨機率等
+ 顯示日出、日落、月出、月落時間
+ 顯示未來一週的天氣預測
+ 可以選擇不同的城市查看天氣資訊


## 技術棧
+ React
+ Sass
+ Fetch API
+ 中央氣象局開放數據 API
